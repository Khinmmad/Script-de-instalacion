# Arch Post Install Script

Script para automatizar la instalación de paquetes después de instalar Arch Linux.

## Uso
```bash
python main.py
```

## Paquetes incluidos
- Paquetes oficiales: firefox, vlc, git...
- AUR: spotify, visual-studio-code-bin...
- Entorno: KDE Plasma
